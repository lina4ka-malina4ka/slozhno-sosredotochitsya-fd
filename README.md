# Проектная работа: адаптивная и резиновая вёрстка + переключение тем

Одностраничный сайт с адаптивной (375 / 768 / 1024+) и “резиновой” вёрсткой, поддержкой светлой/тёмной темы и переключением темы кнопками.

---

## Демо / запуск

Проект статический, достаточно открыть `index.html` в браузере, но удобнее запускать через локальный сервер:

- **VS Code → Live Server**
  - Открой папку проекта
  - ПКМ по `index.html` → **Open with Live Server**

---

## Функциональность

- ✅ Адаптивная вёрстка для 375px / 768px / 1024px+
- ✅ Резиновая вёрстка между брейкпоинтами (grid/flex, fr, clamp и т.д.)
- ✅ Светлая / тёмная тема
- ✅ Поддержка системной темы через `prefers-color-scheme`
- ✅ Переключение темы кнопками в меню
- ✅ Состояния интерактивных элементов (hover/focus-visible)

---

## Технологии

- HTML5 (семантика: `header`, `main`, `section`, `footer`, `nav`)
- CSS3 (Grid, Flex, логические свойства, CSS-переменные, media queries)
- JavaScript (переключение темы)
